Detailed Description

template<class T>
struct hs::util::is_pod< T >

If T is a plain old data type, that is, both trivial and standard-layout, provides the member constant value equal true. For any other type, value is false.

Remarks
This is a simple implementation of std::is_pod using __is_pod from clang builtins.
