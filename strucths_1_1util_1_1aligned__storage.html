Detailed Description

template<size_t Len, size_t Align = sizeof(size_t)>
struct hs::util::aligned_storage< Len, Align >

Obtains a POD type suitable to use as storage for an object of a size of at most Len bytes, aligned as specified by Align.

The obtained type is aliased as member type aligned_storage::type.

Template Parameters
LenThe size of the storage object in bytes.
AlignThe alignment requested in bytes.
Precondition
Len shall not be zero.
Remarks
This is identical to libcxx's aligned_storage.
