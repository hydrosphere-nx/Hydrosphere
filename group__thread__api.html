noexcept

Create a Thread.

Parameters
[in]threadA pointer to a Thread.
[in]thread_entrypointThe entrypoint of the Thread.
[in]argumentThe argument to pass to the entrypoint when starting the Thread.
[in]stackA pointer to a memory region that will be used as a stack by the Thread.
[in]stack_sizeThe size of the stack (must be page aligned).
[in]priorityThe priority of the Thread (0x2C is the usual priority of the main thread. 0x3B on core 0 to 2 and 0x3F on core 3 is a special priority that enables preemptive multithreading).
[in]cpuidThe ID of the CPU core to use (-2 means the default core of the current process).
Precondition
thread state is ThreadState::Uninitialized.
thread_entrypoint is not a null pointer.
stack is not a null pointer.
stack_size is page aligned and not equal to 0.
priority is within the range 0-0x3F.
cpuid is within the range 0-3 or is -2.
Postcondition
thread state is ThreadState::Initialized.

◆ DestroyThread()

void hs::os::DestroyThread ( Thread *  thread)
noexcept

Destroy a Thread.

This function has the following behaviours:

  • If the thread state is ThreadState::Initialized, the thread is started and signaled as ThreadState::Destroyed (The thread will imediately exit).
  • It waits for the Thread to exit and ensures that the Thread stack mirror is unmapped if needed.
Parameters
[in]threadA pointer to a Thread.
Precondition
thread state is not ThreadState::Uninitialized.
Postcondition
thread state is ThreadState::Uninitialized.
